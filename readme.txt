ndbc

Author:  Gregory Dow - cthonianmessiah@gmail.com


Maps a subset of the ODBC API into a node.js plugin.
Includes additional features designed to streamline some common ODBC tasks.

See the source files for detailed documentation.


This was tested on MySQL.  The compiled module is designed to use 32-bit ODBC.
To use it successfully, you must have a 32-bit DSN defined.
